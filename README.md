# Online Shopping Cart

A Java Swing desktop application that simulates a simple online shopping cart system. The application allows users to place product orders through an order form while displaying all submitted orders in a separate window.

---

# Features

## Product Order Management

* Place new product orders
* Generate unique order numbers
* Enter customer information
* Select products from a predefined list
* Specify product quantity
* Display all submitted orders
* Print the order list
* Reset the order form
* Exit the application

---

# Product Categories

The application supports the following product categories:

* Beverage
* Condiment
* Confection
* Dairy Product
* Grains/Cereal
* Meat/Poultry
* Seafood

---

# Order Information

The application records the following details:

* Order Number
* Customer Name
* Product Name
* Quantity Ordered

---

# Order List

Orders are displayed in a separate dialog window using a JTable.

The report includes:

* Order Number
* Customer Name
* Product Name
* Quantity

Features include:

* Automatic order insertion
* Scrollable order table
* Print order list
* Close report window

The application consists of two windows.

* Customer Name

### Order List Window

* Order table
* Print button
* Close button

---

# Application Workflow

1. Launch the application.
2. Enter the customer's name.
3. Select a product category.
4. Enter the quantity.
5. Click **Submit**.
6. A unique order number is automatically generated.
7. The order is displayed in the order list window.
8. Repeat the process for additional orders.
9. Print the order list if required.

---

# How to Run

1. Install JDK 8 or later.
2. Open the project in your preferred Java IDE.
3. Compile the project.
4. Run `MainClass.java`.
